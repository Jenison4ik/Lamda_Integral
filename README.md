Lambda Integral — это интерактивный сервис для тренировки навыков решения интегралов. Выбирайте уровень сложности, решайте задачи и отслеживайте свой прогресс!

## 🚀 Начать работу

Откройте бота в Telegram: **[t.me/lambda_integral_bot](https://t.me/lambda_integral_bot)**

## ✨ Возможности

- **Три уровня сложности**: простой, средний и сложный
- **Интерактивные квизы**: решайте задачи с вариантами ответов
- **Мгновенная обратная связь**: узнавайте правильность ответа сразу
- **Статистика результатов**: отслеживайте свой прогресс
- **Удобный интерфейс**: веб-приложение прямо в Telegram

## 📊 Как это работает

1. **Выберите уровень сложности** — начните с простого или сразу бросьте себе вызов
2. **Решайте задачи** — получайте случайные интегралы для решения
3. **Выбирайте правильный ответ** — из предложенных вариантов
4. **Получайте результат** — в конце квиза увидите статистику своих ответов

## 🎯 Для кого

- Студенты, изучающие математический анализ
- Преподаватели, ищущие дополнительные материалы для практики
- Все, кто хочет улучшить навыки решения интегралов

## 📱 Использование

Просто откройте бота в Telegram и следуйте инструкциям. Всё работает прямо в мессенджере — не нужно устанавливать дополнительные приложения!
